# Helpik 🔍

Проверка купленных номеров телефонов на привязку к кошелькам ЮMoney / Яндекс Pay.

## Использование

1. Введите номера телефонов (по одному на строку)
2. Нажмите "Проверить"
3. Получите результат: чистый номер или привязан чужой кошелёк

## Развёртывание

### Render (бесплатно)
1. Fork этого репозитория
2. Создайте Web Service на [Render](https://render.com)
3. Подключите GitHub репозиторий
4. Готово! Получите URL вида `https://helpik-api.onrender.com`

### Локально
```bash
pip install -r requirements.txt
python app.py
